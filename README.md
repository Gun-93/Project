# CityStyle - E-commerce Project

CityStyle is a **full-stack e-commerce Website** built with **React (Vite)** for the frontend, **Node.js + Express** for the backend, and **MongoDB** as the database.  
It includes user authentication, product listing, cart management, and filtering functionality.
