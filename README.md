# CityStyle - E-commerce Project

CityStyle is a **full-stack e-commerce Website** built with **React (Vite)** for the frontend, **Node.js + Express** for the backend, and **MongoDB** as the database.  
It includes user authentication, product listing, cart management, and filtering functionality.

## 🚀 Features
- 🔐 User authentication (Signup/Login with profile)
- 🛒 Add to cart & checkout system
- 🏷️ Product listing with filters (min/max price)
- 📦 Product details page
- ⚡ Fast frontend built with Vite + React
- 🌐 REST API backend with Express 

---
## 🛠️ Tech Stack
**Frontend:** React (Vite), Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB
